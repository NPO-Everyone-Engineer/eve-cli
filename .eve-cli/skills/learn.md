# 学習モード (Learn Mode)

## 概要
コードやエラーを対話的に解説する学習モード機能です。

## 難易度レベル

| レベル | 解説スタイル | 対象ユーザー |
|--------|-------------|-------------|
| 1 | 超簡潔 | 経験者・要点だけ知りたい |
| 2 | 簡潔 | 中級者・基本的な説明 |
| 3 | 標準 | 一般的な解説（デフォルト） |
| 4 | 詳細 | 初心者・理由も含めて詳しく |
| 5 | 超詳細 | 超初心者・背景知識から丁寧に |

## 機能

### 1. コード生成後の解説
AI がコードを生成した後、「📖 なぜこう書くの？」セクションを自動表示。

### 2. エラー自動解説
エラー発生時に「🔍 なぜエラーが起きたの？」と原因・修正方法を説明。

### 3. インタラクティブ操作
- `y` - わかった（次に進む）
- `?` - もっと詳しく解説
- `s` - この解説をスキップ
- `n` - 解説なしで続ける

## 使用方法

### CLI で起動
```bash
# 学習モードで起動
eve-cli --learn

# 難易度レベルを指定（1-5）
eve-cli --learn --level 3
```

### 対話モード中
```
/learn              # 学習モードのオン/オフを切替
/learn on           # オンに設定
/learn off          # オフに設定
/learn level 4      # レベルを 4 に変更
/learn auto on      # エラー自動解説をオン
/learn auto off     # エラー自動解説をオフ
/learn skip on      # 自動スキップモード
```

## 実装詳細

このスキルは以下の設定を管理します：
- `learn_mode`: 学習モードの有効/無効
- `learn_level`: 解説の難易度レベル（1-5）
- `learn_auto_explain`: エラー自動解説の有効/無効
- `learn_skip_allowed`: 解説スキップの許可/不許可

## 使用例

```bash
# レベル 2 で学習モード開始
eve-cli --learn --level 2

# 対話中にレベル変更
> /learn level 4

# エラー解説だけオフにする
> /learn auto off
```

> **ヒント**: 学習モードは教育現場やコードレビュー、新しい言語を学ぶ際に特に役立ちます。
